Este é um aplicativo Android nativo desenvolvido em **Kotlin** para gestão e controle de estacionamentos. O foco do projeto é oferecer uma interface simples e funcional para operadores gerenciarem o fluxo de veículos e a disponibilidade de vagas.
O app conta com um painel centralizado para acesso rápido às principais funções do pátio.
---

## 🚀 Funcionalidades

O sistema foi desenhado para cobrir as operações essenciais de um estacionamento:

* **Vagas Ocupadas:** Visualização detalhada dos veículos que estão atualmente no pátio.
* **Vagas Livres:** Verificação rápida da disponibilidade para novos clientes.
* **Entrada:** Registro de entrada de novos veículos (Placa, Modelo, Horário).
* **Saída:** Processamento de saída e cálculo de permanência.
* **Histórico:** Log completo de transações e movimentações para auditoria e relatórios.

---

## 🛠️ Tecnologias e Ferramentas

* **Linguagem:** [Kotlin](https://kotlinlang.org/)
* **IDE:** [Android Studio](https://developer.android.com/studio)
* **UI:** XML Layouts (Material Design)
* **Arquitetura:** MVVM (Model-View-ViewModel) - *Em desenvolvimento*
